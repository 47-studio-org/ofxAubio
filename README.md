ofxAubio
========

ofxAubio is an [openFrameworks](openframeworks.cc) [addon](www.ofxaddons.com)
that adds [aubio](aubio.org)

Project Homepage
----------------

The home page of aubio can be found at: http://aubio.org/ofxAubio/
